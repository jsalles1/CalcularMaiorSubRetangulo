# CalcularMaiorSubRetangulo

Projeto em Java para calcular o maior sub-retângulo dentro de uma matriz, utilizando duas abordagens: força bruta e programação dinâmica.

## Visão Geral

***BruteForceSubRetangulo.java***: Implementação de força bruta (ex: verificar todas as possíveis sub-matrizes).

***PDSubRetangulo.java***: Implementação usando Programação Dinâmica (mais eficiente).

Ambos buscam o mesmo objetivo: encontrar qual é o maior sub-retângulo possível, de acordo com o critério definido no problema.

## Como executar
### Clone repositório:
 ```bash
git clone https://github.com/jsalles1/CalcularMaiorSubRetangulo.git
 ```
### Navegue para o diretório:
 ```bash
cd CalcularMaiorSubRetangulo
 ```
### Compile os arquivos Java:
 ```bash
javac BruteForceSubRetangulo.java  
javac PDSubRetangulo.java  
 ```
### Execute o programa com a classe desejada:
 ```bash
java BruteForceSubRetangulo  
# ou  
java PDSubRetangulo 
 ```
